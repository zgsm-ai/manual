# CoStrict购买条款

为规范CoStrict AI编程服务的购买和使用，保障用户和深信服双方的合法权益，特制定本购买条款。请您在购买前仔细阅读本条款。

## 1. 产品套餐及定价

### 1.1 套餐类型

CoStrict目前提供以下Credits流量套餐：

| 套餐名称 | 价格 | Credits数量 | 有效期 |
|---------|------|-------------|--------|
| 流量套餐1 | ¥50 | 1000 Credits | 30天 |
| 流量套餐2 | ¥200 | 4,200 Credits | 30天 |
| 流量套餐3 | ¥500 | 10,800 Credits | 30天 |

### 1.2 Credits消耗规则

- 用户使用不同AI模型时，每次调用消耗的Credits数量不同
- 具体消耗倍率以CoStrict官网实时显示为准
- 深信服有权根据产品运营需要调整各模型的Credits消耗倍率，**无需提前通知用户**
- 用户可在CoStrict客户端或官网查询当前各模型的Credits消耗标准

### 1.3 价格调整

深信服保留调整产品价格的权利，价格调整将在CoStrict官网同步公布，已购买的Credits不受价格调整影响。

## 3. Credits使用规则

### 3.1 有效期

- 所有购买的Credits有效期为**30个自然日**
- 有效期自Credits成功充值到账之日起开始计算
- 到期未使用的Credits将自动失效，不予退还

### 3.2 使用顺序

- 如用户购买多次，Credits将按照到期时间先后顺序消耗
- 即将到期的Credits优先消耗

### 3.3 转让限制

- Credits不可转让、赠予或出售给第三方
- 违反转让限制的，深信服有权冻结相关账户

## 4. 交付异常处理

### 4.1 充值延迟

- 如超过承诺时间未到账，用户可联系客服查询进度
- 客服将在1小时内回复处理结果

### 4.2 账号问题

- 如用户CoStrict账号不存在或被冻结，请先解决账号问题
- 待账号正常后，我们将完成Credits充值

- 深信服有权调整服务内容、功能特性等

- 用户违反服务协议或相关法律法规

- 其他严重违规行为
