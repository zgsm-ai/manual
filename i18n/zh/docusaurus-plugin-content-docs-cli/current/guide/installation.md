# 安装

## CLI 安装

### 前置要求

- Node.js 18.x 或更高版本
- npm 或 yarn

### 通过 npm 安装

```bash
npm install -g @costrict/cli
```

### 通过 yarn 安装

```bash
yarn global add @costrict/cli
```

### 验证安装

```bash
costrict --version
```

### 身份认证

安装完成后，需要进行身份认证：

```bash
costrict login
```

这将打开浏览器窗口，让你使用 CoStrict 账号登录。
