# CLI 功能

CoStrict CLI 提供强大的命令行工具，用于 AI 辅助开发。

## 核心功能

### 交互式聊天模式

启动与 AI 的交互式会话：

```bash
costrict chat
```

### 文件操作

使用 AI 辅助编辑文件：

```bash
costrict edit <文件> --prompt "你的指令"
```

### 代码审查

审查你的代码变更：

```bash
costrict review
```

### 项目分析

分析整个项目：

```bash
costrict analyze
```

## 配置

配置 CLI 设置：

```bash
costrict config set <键> <值>
```

查看当前配置：

```bash
costrict config list
```

## 高级功能

- 自定义提示词和规则
- MCP (模型上下文协议) 集成
- Skills 技能系统
- Plan 模式用于复杂任务

在 [产品特性](../product-features/ai-agent.md) 部分了解更多这些功能。
