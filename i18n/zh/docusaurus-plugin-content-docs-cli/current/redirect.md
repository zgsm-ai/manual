## 计费说明

CLI 和插件使用相同的计费系统。请参考 [插件计费文档](/plugin/billing/purchase)。

### 相关文档

- [购买套餐](/plugin/billing/purchase) - 如何购买和升级套餐

### CLI 特定说明

- CLI 使用计入你账号的总用量
- 你套餐中的所有功能在插件和 CLI 中都可用
- 额度在插件和 CLI 使用之间共享
