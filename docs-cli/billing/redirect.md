# Billing

CLI and Plugin share the same billing system. Please refer to [Plugin Billing Documentation](/plugin/billing/purchase).

## Related Documents

- [Purchase Plans](/plugin/billing/purchase) - How to purchase and upgrade plans
- [Service Description](/plugin/billing/service) - Service tiers and features
- [Usage Explanation](/plugin/billing/usage) - Understanding usage and credits

## CLI-Specific Notes

- CLI usage counts towards your account's total usage
- All features available in your plan work in both Plugin and CLI
- Credits are shared between Plugin and CLI usage
