# Installation

## CLI Installation

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Install via npm

```bash
npm install -g @costrict/cli
```

### Install via yarn

```bash
yarn global add @costrict/cli
```

### Verify Installation

```bash
costrict --version
```

### Authentication

After installation, you need to authenticate:

```bash
costrict login
```

This will open a browser window for you to log in with your CoStrict account.
