# Quick Start

This guide will help you get started with CoStrict CLI.

## Basic Usage

### Start a Chat Session

```bash
costrict chat
```

This opens an interactive chat session with the AI assistant.

### Run a Single Command

```bash
costrict ask "How do I create a React component?"
```

### Work with Files

```bash
costrict edit src/App.js --prompt "Add error handling"
```

## Common Commands

- `costrict chat` - Start interactive chat
- `costrict ask <question>` - Ask a single question
- `costrict edit <file>` - Edit a file with AI assistance
- `costrict review` - Review code changes
- `costrict help` - Show all available commands
