# CLI Features

CoStrict CLI provides powerful command-line tools for AI-assisted development.

## Core Features

### Interactive Chat Mode

Start an interactive session with the AI:

```bash
costrict chat
```

### File Operations

Edit files with AI assistance:

```bash
costrict edit <file> --prompt "your instructions"
```

### Code Review

Review your code changes:

```bash
costrict review
```

### Project Analysis

Analyze your entire project:

```bash
costrict analyze
```

## Configuration

Configure CLI settings:

```bash
costrict config set <key> <value>
```

View current configuration:

```bash
costrict config list
```

## Advanced Features

- Custom prompts and rules
- MCP (Model Context Protocol) integration
- Skills system
- Plan mode for complex tasks

Learn more about these features in the [Product Features](../product-features/ai-agent.md) section.
